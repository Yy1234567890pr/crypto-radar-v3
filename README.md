# 🚀 链上监控雷达 v3 - 极速版

基于 v2 的精准改良，专注速度和成功率。

## ⚡ 核心升级

| 项目 | v2 | v3 |
|------|-----|-----|
| **扫描方式** | 串行 (3-4秒) | **并发** (<1秒) |
| **超时处理** | 默认 | **3秒超时 + 重试** |
| **失败重试** | 无 | **3次指数退避** |
| **备用数据源** | 无 | **DexScreener双保险** |
| **连接复用** | 每次新建 | **HTTP连接池** |
| **预期成功率** | ~70% | **>95%** |
| **扫描时间** | 3-4秒 | **<1秒** |

## 📢 快速开始

```bash
cd ~/crypto-radar-v3
chmod +x start.sh
./start.sh
```

## 🔄 工作流程

1. **并发请求** 4条链同时扫描
2. **智能重试** 593失败自动重试
3. **备用源** GMGN失败时切DexScreener
4. **动量检测** 连涨3轮+涨5% 触发推送
5. **推送** Telegram + 微信同时
